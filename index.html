D3 Financial Charts

A collection of components that make it easy to build interactive financial charts with D3.

Introduction

D3 is a powerful and versatile tool for transforming data into visualisations using SVG, HTML and CSS. One reason why D3 is an ideal tool for creating charts is that you have complete control over how your data is rendered. Every tick mark, gridline, axis and datapoint is constructed manually using primitive elements.

However, with D3 being a low-level tool you have to write quite a bit of code to generate even the simplest of charts.

This is where D3FC comes in! This project provides a set of building blocks that allow you to assemble charts quickly and easily, without taking away the underlying power of D3. These components can be assembled, configured and styled with CSS to produce a whole range of different charts.

Getting Started

Download D3FC

Download the latest JavaScript distribution from GitHub, together with the associated CSS files.

You'll also need to obtain the latest version of D3.

Pick Your Components

Browse the D3FC components to determine which you need in order to build your chart. You can mix an match with any D3 code you may require to produce your chart.

Customise

The components are primarily customised via CSS, allowing you to change colors, stroke, fill and layout. However, more structural customisation is also possible via JavaScript.
